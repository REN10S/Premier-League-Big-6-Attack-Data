# Premier League Big 6 Shooting Analysis

## Overview
This project analyses the attacking performance of the Premier League Big 6 teams using team-level shooting statistics. The focus is on comparing shot volume, accuracy, and finishing efficiency to evaluate attacking effectiveness.

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
Team shooting statistics sourced from FBref (Premier League squad stats).

## Key Metrics
- Shots per 90 (attacking volume)
- Shots on target % (accuracy)
- Goals per shot (finishing efficiency)
- Finishing Index (combined efficiency metric)


## Files
- `PremBig6Attacks.py` – main analysis script
- `PremBig6.csv` – dataset
- `big6_shooting_summary.csv` – processed output
